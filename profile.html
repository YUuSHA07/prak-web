Profil Website

Tentang Website Ini

Website ini dibuat sebagai media pembelajaran dan latihan dalam praktikum Pemrograman Web dengan materi Layout Web Semantik. Tujuan utamanya adalah agar mahasiswa memahami cara membangun struktur halaman web yang baik menggunakan elemen-elemen semantik seperti <header>, <nav>, <article>, <section>, <aside>, dan <footer>.

Visi

Menjadi platform pembelajaran web yang interaktif dan edukatif, sehingga mahasiswa dapat menguasai dasar-dasar pengembangan web modern sesuai dengan standar HTML5, CSS3, dan praktik semantik yang baik.

Misi

  • Meningkatkan pemahaman mahasiswa terhadap konsep Semantic HTML.
  • Memberikan contoh penerapan layout web yang terstruktur dan responsif.
  • Mengintegrasikan teori dan praktik dalam proses pembelajaran web development.
  • Membantu mahasiswa mengenali standar penulisan kode yang rapi dan mudah dibaca.
